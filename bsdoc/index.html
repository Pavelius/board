Описание формата

Все конфигурационные файлы в игре имеют расширение *.txt и это обычные текстовые файлы. Каждый такой файл содержит список объектов и их свойств.

Начиная с новой строки указывается символ "#" после которого сразу без пробела следует тип объекта, который должен быть одним из указанных ниже типов. Это идентификатор, начинающийся с буквы или знака подчеркивания, который внутри себя не содержит пробелов, но может содержать цифры, буквы и знаки подчеркивания.

После идентификатора имени идет ключевое поле обекта. Обычно это свойство с именем id имеющее тип "строка". По ключевому полю будет произведен поиск объекта в таблице и если до этого объект такого же типа с таким же ключевым поле был объявлен дальнейшее описание свойств будет касаться этого объекта. Иначе будет создан новый объект с таким ключевым полем.

После ключевого поля перечисляются заполненные свойства объекта в формате:

идентификатор поля=(значение).

Некоторые типы объектов имеют длинное, часто многострочное, текстовое описание, которое может выводится, например, в виде выпадающей подсказки при наведении на него мышкой. Это описание можно ввести после всех свойств в виде текста с новой строки. Оконанием такого блока будет начало следуюего блока объекта, то есть наало строки с символа "#" и следующего за ним идентификатора. Весь этот блок преобразуется в строку и заносится в свойство text.
