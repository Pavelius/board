Описание формата файла

Каждая строка файла содержит описание одного объекта.

Каждое описание сущности начинает строку с символа '#' сразу за которым следует идентификатор типа объекта.

Некоторые из объектов могут иметь описание, которое может отображаться в подсказке или других элементах, содержащих много текста. Эта специальная строка текста добавляется сразу после описания объекта и перед следующим объектом.
