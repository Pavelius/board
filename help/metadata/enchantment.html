Enchantment

Этот объект используется для отображения enchantment.

Синтаксис

#enchantment id name(строка) attack(число) defend(число) raid(число)
text

Описание полей

Имя поляТипОписание
idстрокаИспользуйте этот идентификатор, когда хотите указать ссылку на данный объект.
nameстрокаНаименование, используемое для отображения объекта.
textстрокаОписание объекта. Начинается с новой строки следующей сразу за объектом.
attackчисло
defendчисло
raidчисло
